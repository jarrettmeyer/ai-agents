# LangGraph OpenAI ChatBot

- [LangGraph](https://langchain-ai.github.io/langgraph/)
- [Tavily Search](https://tavily.com/)

1. Make a a new `.env` file.
    ```bash
    cp .env.example .env
    ```

2. Install required packages.
    ```bash
    pip install -r requirements.txt
    ```

3. Run the chatbot.
    ```bash
    python chatbot.py
    ```
